# CodeAlpha Web Scraping

## 📌 Project Overview
This project is developed as part of the CodeAlpha Data Analytics Internship. It demonstrates web scraping using Python to collect book information from a public website and save it as a CSV dataset.

## 🎯 Objective
- Extract book details from a website
- Create a structured dataset
- Save the extracted data in CSV format

## 🛠️ Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas

## 📂 Project Files
- web_scraping.py
- books_dataset.csv

## 📊 Output
The program extracts:
- Book Title
- Price
- Availability

and stores the data in `books_dataset.csv`.

## 🌐 Website Used
https://books.toscrape.com/
